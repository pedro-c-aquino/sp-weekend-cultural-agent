# sp-weekend-cultural-agent
Multi‑agent ReAct + (optional) RAG system that plans, searches the web, and extracts up‑to‑date cultural events (plays, expositions, musical shows—especially samba) in São Paulo for the current weekend. Built with Python + LangChain. Default stack prefers open‑source (Ollama + sentence‑transformers + duckduckgo-search + Chroma).
